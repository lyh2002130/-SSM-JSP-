# 基于SSM+JSP的花店项目

#### 介绍
基于Idea开发的有间花店项目，非常适合实战练习项目和毕业设计项目

#### 项目技术
Spring+ SpringMVC + Mybatis + JSP +Maven

#### 安装教程
1.  Mysql5.7
2.  Maven
3.  JDK1.8

#### 使用说明
1.  运行SQl文件创建数据库
2.  配置一下Tomcat服务器
3.  然后直接运行
